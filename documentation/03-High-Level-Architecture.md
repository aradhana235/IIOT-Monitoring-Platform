# Enterprise Industrial IoT Monitoring Platform

# Phase 3 – High Level Architecture

## Objective

Design a scalable, secure, and maintainable architecture for an Industrial IoT Monitoring Platform capable of handling telemetry data from thousands of industrial devices.

---

## Architecture Overview

The system consists of the following major components:

- React Frontend
- Spring Boot Backend
- PostgreSQL
- Apache Cassandra
- TTN (The Things Stack)
- WebSocket
- JWT Authentication
